# Inbound Content — 内容模板

## 信任型内容（建立专业度）

### 模板 1：案例拆解
```
标题：[客户类型] 如何用 GEO 实现 [具体结果]

结构：
1. 客户背景（痛点）
2. 优化前数据
3. 具体做法（3-5 步）
4. 优化后数据
5. 关键洞察（1-2 句）

CTA：需要 GEO 落地方案？DM 我聊聊。
```

### 模板 2：方法论
```
标题：[数字] 步搞定 [GEO 具体问题]

结构：
1. 问题说明（为什么重要）
2. 步骤 1-5（每步具体操作）
3. 常见坑（1-2 个）
4. 工具推荐（可选）

CTA：整理成 PDF 清单了，需要 DM 我。
```

## 钩子型内容（引导行动）

标题：我整理了 [资源名]，包含 [具体内容]

内容：
- [资源亮点 1]
- [资源亮点 2]
