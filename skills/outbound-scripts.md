# Outbound Scripts — DM 话术库

## 模板 A：价值先行（P0 目标）

```
Hi [名字], 看到你 [具体问题/需求]。

我们刚帮 [类似公司] 解决了这个问题，[具体结果，如"GEO 优化后 AI 引用率提升 3 倍"]。

如果你有兴趣，我发案例给你看看？

（不用回复"不用"，直接说"发我"就行）
```

## 模板 B：钩子 CTA（P1 目标）

```
Hi [名字], 注意到你在关注 [GEO 相关话题]。

我们整理了一份《GEO 落地清单》，包含 12 个具体步骤和 5 个工具推荐。

需要的话 DM 我，免费发你参考。
```

## 模板 C：共情 + 方案（跟进用）

```
[名字], 理解你在 [痛点] 上的困扰。

[类似客户] 之前也这样，后来我们用 [方法] 解决了，[结果]。

有兴趣花 15 分钟聊聊吗？我分享具体做法。
```
