# DM Interaction Patterns

## 高回复率话术模式

### 1. 价值先行
```
看到你 [具体问题]，我们刚帮 [类似公司] 解决了这个问题，
[具体结果]。如果你有兴趣，我发案例给你看看？
```

### 2. 钩子 CTA
```
我们整理了一份 [具体清单/模板]，包含 [数量] 个 [具体内容]。
需要的话 DM 我，免费发你。
```

### 3. 共情 + 方案
```
理解你的痛点。[类似客户] 之前也这样，后来我们用 [方法] 解决了。
有兴趣聊聊吗？
```

---

## 跟进节奏

| 时间 | 动作 | 话术类型 |
|------|------|---------|
| Day 0 | 首次 DM | 价值先行 + 钩子 |
| Day 3 | 跟进 1 | 换角度 + 新价值 |
| Day 6 | 放弃 | 不再跟进 |
